BEBZOCOURSE V3

LANCEMENT WINDOWS
1. Dézippe le dossier
2. Double-clique sur LANCER_BEBZOCOURSE.bat
3. Laisse la fenêtre noire ouverte
4. Ferme-la pour arrêter l'application

NOUVEAUTÉS
- Finito les courses : ajoute la course à l'historique
- statistiques, graphique mensuel et calendrier de consommation
- favoris automatiques après 3 courses consécutives
- rappels intelligents optionnels
- suggestions de recettes optionnelles
- tri de Ma liste : A-Z, rayons, catégories ou aucun tri
- onglet Saisonnier avec calendrier annuel
- tags De saison optionnels
- recherche rapide avec Entrée
- export/import des données

TÉLÉPHONE HORS LIGNE
Cette application est une PWA. Pour l'installer sur Android, elle doit être ouverte une première fois depuis une adresse HTTPS, puis Chrome > Installer l'application. Une fois installée et chargée, elle fonctionne hors ligne. Les données restent locales à chaque appareil.
